# Flappy Birb 🐦

A Flappy Bird clone made with Python and Pygame.

## How to Play
- Press **Space** or **Click** to flap
- Avoid the pipes
- Every pipe, the gap gets smaller :3

## Running from Source
Make sure you have Python installed, then:
```
pip install pygame
python flappy.py
```

## Running from .exe
```
Download .exe from the releases
and play
enjoy ;3
```

## Controls
| Input | Action |
|-------|--------|
| Space | Flap |
| Mouse Click | Flap |
